# In a production environment, where is the most ...

## The Question
In a production environment, where is the most secure place to store your Anthropic API key?

### Options
- A) Within the client-side JavaScript of your web application.
- B) In a public repository’s .env.example file.
- C) On a server-side environment variable, hidden from the end-user.
- D) Within the local storage of the user's web browser.

## Answer
C) On a server-side environment variable, hidden from the end-user.

## Explanation
API keys should never be exposed on the client side. They should be stored in secure server-side environment variables.
