## The Question
You need Claude to return a valid JSON object without any conversational "filler" text. Which technique is most reliable?

### Options
- A) Use high temperature and ask "Please be brief."
- B) Use a system prompt and XML tags for the output.
- C) Use a prefilled assistant message and stop sequences.
- D) Use a few-shot prompt with 10 examples of text-heavy JSON.

## Answer
C) Use a prefilled assistant message and stop sequences.

## Explanation
Prefilling the assistant's response with '{' forces the model to start the JSON object immediately, minimizing conversational filler.
