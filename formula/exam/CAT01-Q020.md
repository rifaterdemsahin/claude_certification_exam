# Exam Question CAT01-Q020

## Question
You want to use Claude 3.7 Sonnet’s "Extended Thinking" feature. What should you expect in the API response?

## Options
- A) A hidden internal log that is only accessible via the Anthropic Console.
- B) A response containing both the reasoning process and the final answer.
- C) A significant reduction in token usage for complex math problems.
- D) A set of Python code blocks that the model executed internally.

## Answer
B

## Explanation
Extended Thinking allows the model to output its internal reasoning process alongside the final response.
