# Exam Question CAT01-Q010

## Question
What is the minimum token count required for a content block to be eligible for Prompt Caching?

## Options
- A) 512 tokens
- B) 1024 tokens
- C) 2048 tokens
- D) 4096 tokens

## Answer
B

## Explanation
Anthropic's Prompt Caching requires blocks to be at least 1024 tokens to be eligible for caching (on supported models).
