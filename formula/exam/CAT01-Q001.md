# Exam Question CAT01-Q001

## Question
What is the primary function of a "tool" in the context of Claude’s API?

## Options
- A) To automatically compress conversation history to save tokens.
- B) To provide a configuration file that dictates Claude's persona.
- C) A plain function executed when Claude needs external data or to perform an action.
- D) An internal Anthropic database query for user preferences.

## Answer
C

## Explanation
Tools (also known as function calling) allow Claude to interact with external systems by defining functions it can request to execute.

## Deep Dive
Tools are defined using JSON Schema. When Claude 'calls' a tool, it provides the arguments as a JSON object. The client application is then responsible for executing the function and returning the result. This enables Claude to perform real-world actions like searching the web, querying databases, or manipulating files.
