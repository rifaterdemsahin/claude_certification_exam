# Exam Question CAT01-Q038

## Question
When using Claude Code, what happens if a request matches the description in a skill's frontmatter?

## Options
- A) Claude asks the user for permission to use the skill.
- B) The skill is automatically loaded into the conversation context.
- C) The skill replaces the CLAUDE.md file temporarily.
- D) Claude ignores the skill if a slash command is also available.

## Answer
B

## Explanation
Claude Code uses a routing mechanism that identifies and loads the most relevant skills based on their descriptions.
