# Exam Question CAT01-Q023

## Question
You are building a high-volume support bot. You use the same 5,000-token instruction set for every customer query. Which feature provides the most cost optimization?

## Options
- A) PDF processing
- B) Prompt Caching
- C) Response Streaming
- D) Model Grading

## Answer
B

## Explanation
Prompt Caching allows you to reuse large blocks of text (like system instructions) across many requests at a significantly reduced cost.
