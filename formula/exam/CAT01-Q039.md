# Exam Question CAT01-Q039

## Question
You are building a RAG (Retrieval-Augmented Generation) pipeline. Where does Claude fit in?

## Options
- A) Claude is the vector database where documents are stored.
- B) Claude is the reasoner that uses retrieved text (provided via prompt or tool) to answer questions.
- C) Claude is the embedding model that converts text to numbers.
- D) Claude is the web crawler that finds the documents.

## Answer
B

## Explanation
In RAG, Claude acts as the cognitive engine that processes the retrieved information to generate a grounded response.
