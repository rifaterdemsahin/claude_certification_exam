# Exam Question CAT01-Q021

## Question
When designing an MCP resource for a documentation system, what is the best way to handle dynamic paths like docs://manual/chapter-1?

## Options
- A) Create a unique static resource for every possible chapter.
- B) Use a templated resource with parameters in the URI.
- C) Use a Tool instead of a Resource for all document access.
- D) Hardcode the documents into the MCP server's prompt.

## Answer
B

## Explanation
MCP resource templates allow for dynamic URI schemes that can map to various data points based on parameters.

## Deep Dive
Model Context Protocol (MCP) is an open standard that enables AI models to interact with data and tools from various sources. It uses a client-server architecture where servers expose 'primitives' like tools, resources, and prompts. This modularity allows for highly scalable and maintainable AI integrations.
