# Exam Question CAT01-Q026

## Question
You are designing a complex orchestration involving a "Planner" model and several "Worker" models. Which pattern is this?

## Options
- A) Single-prompt chain
- B) Parallel sub-agents (Orchestrator-Workers)
- C) Simple linear workflow
- D) Prompt Caching hierarchy

## Answer
B

## Explanation
The Orchestrator-Workers pattern uses a central planner to delegate tasks to multiple specialized sub-agents.
