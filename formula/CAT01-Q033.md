## The Question
You want Claude to reference a specific local file in a conversation without uploading it every time. Which feature (currently in preview) allows this?

### Options
- A) Tool Use
- B) Files API
- C) Prompt Caching
- D) MCP Resources

## Answer
D) MCP Resources

## Explanation
MCP Resources allow an AI client to access external data points (like local files) through a standardized interface.

Building reliable applications with Claude requires a blend of traditional software engineering principles and AI-specific techniques. Security is paramount; API keys must always be stored on the server side, and any code execution should happen in a sandboxed environment. Performance optimization, such as using streaming for better user experience or batching tool calls to reduce round-trips, is essential for production-grade apps.

The evaluation of AI systems is an ongoing process. Instead of relying on manual vibes-based testing, developers should implement evaluation workflows using diverse datasets and model-based graders.
