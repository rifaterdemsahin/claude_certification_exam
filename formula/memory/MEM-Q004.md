# Memory Card MEM-Q004

## Question
What is Session Management in agentic architecture?

## Answer
Maintaining state and context across multiple interactions or task executions within a single user session.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
