# Memory Card MEM-Q010

## Question
What is the purpose of routing logic in the Coordinator pattern?

## Answer
To determine which sub-agent is best suited to handle a specific task or query.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
