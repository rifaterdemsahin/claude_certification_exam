# Memory Card MEM-Q029

## Question
What is the key principle of MCP as 'Protocol over API'?

## Answer
MCP standardizes how tools expose capabilities, allowing you to swap data sources without rewriting agent logic.

## Deep Dive
The Model Context Protocol (MCP) provides a standardized way for AI models to access external data. This decoupling of the model from the data source enables greater flexibility and reuse of tools.
