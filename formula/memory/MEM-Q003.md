# Memory Card MEM-Q003

## Question
What is Task Decomposition in agentic systems?

## Answer
Breaking complex tasks into smaller, manageable subtasks that can be assigned to different agents or executed sequentially.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
