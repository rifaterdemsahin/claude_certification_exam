# Memory Card MEM-Q022

## Question
What is a 'feedback loop' in agentic systems?

## Answer
The mechanism by which the agent receives feedback on its actions and adjusts future behavior.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
