# Memory Card MEM-Q071

## Question
What is 'chain-of-thought prompting'?

## Answer
Prompting the model to show its reasoning step-by-step before giving the final answer.

## Deep Dive
Advanced prompt engineering involves more than just instructions; it requires structured output formats like JSON and validation loops to ensure the AI's response meets technical requirements.
