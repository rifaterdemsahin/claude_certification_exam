# Memory Card MEM-Q039

## Question
What is a 'tool fallback strategy'?

## Answer
Having a backup plan when a primary tool fails, such as using cached data or an alternative service.

## Deep Dive
The Model Context Protocol (MCP) provides a standardized way for AI models to access external data. This decoupling of the model from the data source enables greater flexibility and reuse of tools.
