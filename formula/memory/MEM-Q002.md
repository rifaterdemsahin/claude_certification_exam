# Memory Card MEM-Q002

## Question
What is the Coordinator-Subagent model?

## Answer
A pattern where a central coordinator agent routes tasks to specialized sub-agents rather than using one massive prompt.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
