# Memory Card MEM-Q027

## Question
What is 'human-in-the-loop' orchestration?

## Answer
A design where humans review, approve, or provide input at critical decision points in the agent workflow.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
