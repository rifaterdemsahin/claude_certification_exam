# Memory Card MEM-Q007

## Question
What does 'stateless' mean in agent design?

## Answer
Each request is independent with no memory of previous interactions; state must be explicitly passed or stored.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
