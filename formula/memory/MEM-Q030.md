# Memory Card MEM-Q030

## Question
What is 'Signal-to-Noise' ratio in tool interfaces?

## Answer
Designing tools to return highly relevant, structured data rather than overwhelming the agent with irrelevant information.

## Deep Dive
The Model Context Protocol (MCP) provides a standardized way for AI models to access external data. This decoupling of the model from the data source enables greater flexibility and reuse of tools.
