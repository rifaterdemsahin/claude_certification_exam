# Memory Card MEM-Q005

## Question
What is the difference between a simple prompt and an agentic loop?

## Answer
A simple prompt is linear (input → output), while an agentic loop is cyclic (observe → reason → act → observe).

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
