# Memory Card MEM-Q026

## Question
What is a 'termination condition' for an agent?

## Answer
The criteria that signal the agent should stop its loop, such as task completion or max iterations.

## Deep Dive
Agentic architecture focuses on the cyclic nature of AI interaction. By implementing loops, agents can self-correct and handle complex, multi-step tasks that linear prompts cannot.
