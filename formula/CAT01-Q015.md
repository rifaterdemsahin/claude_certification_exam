## The Question
In Claude Code, what is the primary difference between CLAUDE.md and a Skill?

### Options
- A) CLAUDE.md is for personal use, while Skills are for teams.
- B) CLAUDE.md loads into every conversation, while Skills load on-demand based on the task.
- C) Skills require a paid subscription, while CLAUDE.md is free.
- D) CLAUDE.md defines tools, while Skills only define system prompts.

## Answer
B) CLAUDE.md loads into every conversation, while Skills load on-demand based on the task.

## Explanation
CLAUDE.md provides base context for every session, while Skills are specialized and activated when needed.
