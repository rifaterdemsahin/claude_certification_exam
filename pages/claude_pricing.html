Evolutionary Trends

1. The Tiered Strategy: In 2023, there was only "Claude" and "Claude Instant". With Claude 3, Anthropic moved to the three-tier model we see today.

2. Context vs. Cost: While pricing for flagship models initially increased (Claude 2 at $8 -> Claude 3 Opus at $15), the context window expanded from 9K (early Claude 1) to 200K (standard today).

3. Efficiency Gains: Claude 3.5 Sonnet offers performance superior to Claude 3 Opus but at the 80% lower price point of the Sonnet tier.

4. Pricing Adjustments: Claude 3.5 Haiku saw a price increase compared to 3.0 Haiku ($0.80 vs $0.25), reflecting its significant jump in intelligence (surpassing Claude 3 Opus on some benchmarks).

Prompt Caching (The New Frontier)

Introduced in late 2024, Prompt Caching allows for massive savings on repetitive context:

  • Cache Write: 25% surcharge on base input price.
  • Cache Read: ~90% discount on base input price.

This fundamentally changes the economics of agentic loops that reuse large system prompts or codebase context.
