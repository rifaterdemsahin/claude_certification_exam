# Claude Developer Certification - Study Mastery App

An interactive study guide for the **Claude Developer Certification** exam with 100 questions, visual hints, memory aids, and progress tracking.

## 🚀 Live Demo

**[Study App - GitHub Pages](https://rifaterdemsahin.github.io/claude_certification_exam/)**

## 📋 Features

- **100 Exam Questions** covering all 5 core competencies
- **Memory Cards** with flip-to-reveal answers
- **Visual Hints** with SVG graphics and emoji mnemonics
- **Memory Aids** to help you remember key concepts
- **Quiz Mode** with multiple choice answers
- **Progress Tracking** saved in browser cookies
- **Learning Resources** with curated YouTube videos
- **Category Filtering** by competency area

## 🛠️ Technologies

- React 18 (via CDN)
- Vanilla CSS with dark theme
- Cookie-based state management
- Single-file application (no build step required)
