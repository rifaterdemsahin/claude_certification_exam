# 🔗 Official Bookmarks & Study Resources

This document maintains a curated directory of all official documentation, learning platforms, developer consoles, and hosting environments required to study for and build tools for the **Claude Developer Certification**.

---

## 🤖 1. Official Anthropic Developer Portal
* **Anthropic Homepage:** [anthropic.com](https://www.anthropic.com) — Corporate announcements and certification details.
* **Anthropic Developer Docs:** [docs.anthropic.com](https://docs.anthropic.com) — Standard documentation for building with Claude.
* **Anthropic Console:** [console.anthropic.com](https://console.anthropic.com) — Manage API keys, billing, and test prompts in the Workbench.
* **Anthropic Pricing Tiers:** [anthropic.com/pricing](https://www.anthropic.com/pricing) — Official up-to-date pricing sheet for Claude models (Haiku, Sonnet, Opus).
* **Anthropic Skilljar Portal:** [anthropic.skilljar.com](https://anthropic.skilljar.com) — Learning pathways and training portal for official exams.

---

## 🔌 2. Model Context Protocol (MCP) Bookmarks
* **MCP Specification Site:** [modelcontextprotocol.org](https://modelcontextprotocol.org) — Main hub for Model Context Protocol standards.
* **MCP GitHub Organization:** [github.com/modelcontextprotocol](https://github.com/modelcontextprotocol) — Core SDK repos, TypeScript/Python implementations, and starter templates.
* **Standard MCP Servers Repository:** [github.com/modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers) — Reference implementations for standard servers (Postgres, Git, Filesystem, Google Drive, Slack).

## 🛠️ 3. Claude Code & CLI Links
* **Claude Code Documentation:** [docs.anthropic.com/en/docs/agents-and-tools/claude-code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code) — Setup, interactive commands, slash commands (`/search`, `/bug`), and plan/act mode guide.
