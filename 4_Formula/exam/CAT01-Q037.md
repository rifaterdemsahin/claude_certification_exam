# Exam Question CAT01-Q037

## Question
Which MCP feature is specifically designed to provide high-quality, pre-tested instructions for a common task?

## Options
- A) Resources
- B) Tools
- C) Prompts
- D) Context Blocks

## Answer
C

## Explanation
MCP Prompts are server-side templates that provide a standard way to interact with tools and resources.

## Deep Dive
Model Context Protocol (MCP) is an open standard that enables AI models to interact with data and tools from various sources. It uses a client-server architecture where servers expose 'primitives' like tools, resources, and prompts. This modularity allows for highly scalable and maintainable AI integrations.
