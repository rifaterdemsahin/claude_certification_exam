# Exam Question CAT01-Q007

## Question
Which Claude Code customization feature is best for a team-wide PR review checklist that should be shared via Git?

## Options
- A) A personal skill in ~/.claude/skills.
- B) A project-specific skill in .claude/skills.
- C) A global alias in the user's .zshrc.
- D) A custom slash command defined only in the local terminal.

## Answer
B

## Explanation
Project-specific skills are stored in the project directory, making them easy to share with team members via version control.
