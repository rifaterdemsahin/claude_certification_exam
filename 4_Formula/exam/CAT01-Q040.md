# Exam Question CAT01-Q040

## Question
You want to reduce the "time to first token" for a user-facing chat app so it feels more responsive. What should you implement?

## Options
- A) Prompt Caching
- B) Response Streaming
- C) Batch Tooling
- D) Model Grading

## Answer
B

## Explanation
Streaming allows you to display parts of the model's response as they are generated, rather than waiting for the entire block.
