# Exam Question CAT01-Q032

## Question
Which of these is a required parameter for a successful messages API call?

## Options
- A) user_id
- B) max_tokens
- C) temperature
- D) stop_sequences

## Answer
B

## Explanation
The Anthropic Messages API requires 'model', 'messages', and 'max_tokens' for every request.
