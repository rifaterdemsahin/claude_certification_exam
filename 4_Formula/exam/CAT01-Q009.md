# Exam Question CAT01-Q009

## Question
When using the Model Context Protocol (MCP), what is the role of an MCP Server?

## Options
- A) To manage user authentication and billing for the end application.
- B) To host tools, prompts, and resources that Claude can access.
- C) To act as the primary user interface for the AI chat.
- D) To route messages between different LLM providers.

## Answer
B

## Explanation
MCP Servers expose capabilities (Tools), data (Resources), and templates (Prompts) to AI clients.

## Deep Dive
Model Context Protocol (MCP) is an open standard that enables AI models to interact with data and tools from various sources. It uses a client-server architecture where servers expose 'primitives' like tools, resources, and prompts. This modularity allows for highly scalable and maintainable AI integrations.
