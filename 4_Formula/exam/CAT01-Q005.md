# Exam Question CAT01-Q005

## Question
You notice a prompt is failing on edge cases in production. What is the most effective next step?

## Options
- A) Increase the temperature to 1.0 to bypass logic errors.
- B) Implement a prompt evaluation workflow with a diverse dataset.
- C) Manually rewrite the prompt until it feels more comprehensive.
- D) Switch to a smaller model like Claude 3 Haiku to save costs.

## Answer
B

## Explanation
Prompt evaluation with a test suite/dataset is the standard engineering approach to ensure reliability and handle edge cases.
