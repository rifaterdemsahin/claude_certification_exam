# Exam Question CAT01-Q028

## Question
If you want Claude to be as deterministic and predictable as possible for a data extraction task, what temperature should you set?

## Options
- A) 1.0
- B) 0.5
- C) 0.0
- D) -1.0

## Answer
C

## Explanation
Setting temperature to 0.0 minimizes randomness, making the model's outputs more deterministic.
