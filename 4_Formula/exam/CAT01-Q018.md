# Exam Question CAT01-Q018

## Question
What does the "Batch Tool" (or computer_use style multi-calls) primarily solve in orchestration?

## Options
- A) It allows Claude to run code in an un-sandboxed environment.
- B) It reduces latency by allowing Claude to issue multiple tool calls in a single turn.
- C) It increases the accuracy of mathematical calculations.
- D) It encrypts tool parameters before they are sent to the server.

## Answer
B

## Explanation
Batching tool calls allows the model to request several actions at once, reducing the number of round-trips needed.

## Deep Dive
Tools are defined using JSON Schema. When Claude 'calls' a tool, it provides the arguments as a JSON object. The client application is then responsible for executing the function and returning the result. This enables Claude to perform real-world actions like searching the web, querying databases, or manipulating files.
