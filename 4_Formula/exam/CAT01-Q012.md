# Exam Question CAT01-Q012

## Question
When developing an MCP server using the Python SDK, what is the most efficient way to define a tool?

## Options
- A) Manually writing a JSON schema for every tool function.
- B) Using the @mcp.tool() decorator on a Python function.
- C) Registering the tool via a separate YAML configuration file.
- D) Using a POST request to the MCP discovery endpoint.

## Answer
B

## Explanation
The MCP Python SDK provides decorators like @mcp.tool() to automatically generate the necessary JSON schemas from Python function signatures.

## Deep Dive
Tools are defined using JSON Schema. When Claude 'calls' a tool, it provides the arguments as a JSON object. The client application is then responsible for executing the function and returning the result. This enables Claude to perform real-world actions like searching the web, querying databases, or manipulating files.
