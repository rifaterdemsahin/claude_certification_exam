# Exam Question CAT01-Q013

## Question
Which scenario describes a "Workflow" pattern rather than an "Agent" pattern?

## Options
- A) Claude is given a set of tools and decides which ones to use to solve an open-ended goal.
- B) Claude follows a predefined, rigid sequence of steps to process an invoice.
- C) Claude manages a long-running project by assigning tasks to sub-agents.
- D) Claude proactively monitors a database and alerts users of anomalies.

## Answer
B

## Explanation
Workflows are structured, predefined sequences, whereas agents are more autonomous and decide their own path toward a goal.

## Deep Dive
Agentic patterns differ from simple workflows by their degree of autonomy. Agents use a loop (e.g., Perceive-Think-Act) to iteratively work towards a goal, making decisions about which tools to use and how to handle unexpected outputs or errors.
