# Exam Question CAT01-Q006

## Question
What does "transport agnostic" mean regarding the Model Context Protocol (MCP)?

## Options
- A) MCP only functions over encrypted HTTPS connections.
- B) Clients and servers can connect via different methods like stdio or HTTP.
- C) MCP requires a specific hardware acceleration chip to function.
- D) The protocol automatically switches to the fastest available network.

## Answer
B

## Explanation
Transport agnostic means the protocol defines how messages are structured, but can work over various communication channels like stdio, HTTP, or WebSockets.

## Deep Dive
Model Context Protocol (MCP) is an open standard that enables AI models to interact with data and tools from various sources. It uses a client-server architecture where servers expose 'primitives' like tools, resources, and prompts. This modularity allows for highly scalable and maintainable AI integrations.
