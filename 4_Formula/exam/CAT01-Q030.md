# Exam Question CAT01-Q030

## Question
You are using Claude Code and want it to follow specific brand guidelines (colors, fonts) only when you are working on the "frontend" directory. Where should you put these instructions?

## Options
- A) In a global skill in ~/.claude/skills.
- B) In a project skill in .claude/skills/brand-style/SKILL.md.
- C) In the root CLAUDE.md.
- D) In the .gitignore file.

## Answer
B

## Explanation
Project-specific skills allow for directory-scoped instructions and tools.
